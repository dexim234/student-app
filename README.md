# ApeVault Students App

Сайт для учеников ApeVault.

## Разделы

- **Личный кабинет** - Профиль ученика (в разработке)
- **Кошелек** - Управление балансом (в разработке)
- **Club** - Лента торговых сигналов от трейдеров команды
- **Торговля** - Торговая платформа (в разработке)
- **Рефералы** - Реферальная система (в разработке)
- **Настройки** - Настройки аккаунта (в разработке)

## Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно на `http://localhost:5174`

## Сборка

```bash
npm run build
```

## Структура

- `src/pages/` - Страницы приложения
- `src/components/` - React компоненты
- `src/store/` - Zustand stores для управления состоянием
- `src/services/` - Сервисы для работы с Firebase
- `src/types/` - TypeScript типы
- `src/firebase/` - Конфигурация Firebase
